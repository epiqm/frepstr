frepstr
=======

Utility for replacing text in multiple files on specified folder (including sub-folders)

## Usage

```sh
$ ./frepstr -h
File replace string

Usage:
	$ ./frepstr 
		-dir[=./]       working directory
		-f,find="something" 
		-r,replace="different"
```
